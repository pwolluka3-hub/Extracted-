# App Agent Skill Sources

The built-in app agent skills were expanded using role/pattern scans from public GitHub prompt libraries:

- `f/awesome-chatgpt-prompts` (`prompts.csv`)
  - Relevant patterns used: Social Media Manager, Social Media Influencer, LinkedIn Ghostwriter, YouTube Video Analyst, Creative Branding Strategist, SEO specialist.
- Additional cross-checks on prompt-library ecosystem repos for content operations and social workflows.

These source patterns were converted into product-safe app skills focused on:

- content creation and repurposing
- platform-native adaptation
- community management and response drafting
- analytics feedback and A/B variation
- media production planning (reels/carousels/scripts)
- brand safety, quality consistency, and execution speed
